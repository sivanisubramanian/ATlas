# ATlas — Athletic Training Learning System

A structured learning platform built for student athletic trainers at the high school level and beyond.

## The Problem

Student ATs are expected to learn on the job, but supervising ATs are too busy to hand-hold and there's no single resource that tells students what to learn or in what order. ATlas solves this with a structured curriculum, interactive lessons, and a quick-reference sideline tool.

## What's Inside

**Level 1 — Foundational Information for Club Members**
**Level 2 - Basic Skills for JV ATs**
**Level 3 - Advanced Information for Varsity ATs**
**Quick Reference**
A fast-lookup page for sideline use: concussion red flags, heat illness spectrum, when to call 911, wound care steps, taping notes, and common injury signs.

## Tech

Single self-contained HTML file. No frameworks, no dependencies, no build step. All images are base64-encoded inline.

## Run Locally

Download `index.html` and open it in any browser. That's it.

## Deploy to Netlify

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop this folder
3. Live instantly

To auto-deploy on every push, connect this repo to Netlify via **Add new site → Import an existing project**.
